# React Native Calendar App

This project is a mobile web app built with React Native. It features a login page based on the provided Figma design, with email and password fields, and a sign up button styled as in the second Figma link. The app is prepared for future calendar and event management features.

## Features
- Login screen (no Google/Facebook buttons)
- Sign up button (custom style)
- Ready for calendar/event features

## Getting Started
1. Install dependencies:
   ```
   npm install
   ```
2. Run the app:
   ```
   npx react-native run-android # or run-ios
   ```

## Folder Structure
- App.js / App.tsx: Main entry point
- components/: UI components
- screens/: App screens (Login, SignUp, etc.)

## Notes
- All UI is based on the provided Figma links.
- Only React Native is used (no Expo).
- Placeholder assets should be replaced with actual images/icons from Figma.
